# The Quant Society

Official website for The Quant Society - a student organization dedicated to exploring quantitative finance, data science, and analytics.

## About

The Quant Society brings together passionate students interested in mathematical modeling, algorithmic trading, risk management, and data-driven decision making.

## Website Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional interface with smooth animations
- **Interactive Navigation**: Smooth scrolling and mobile-friendly hamburger menu
- **Sections**:
  - Home: Hero section with call-to-action buttons
  - About: Overview of the society and key focus areas
  - Events: Upcoming workshops, lectures, and hackathons
  - Team: Leadership and organizational structure
  - Resources: Learning materials, tools, and recommended books
  - Contact: Contact form and meeting information

## How to View

Simply open `index.html` in your web browser to view the website.

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript (no dependencies)

## Customization

To customize the website for your college:

1. Update contact information in the Contact section
2. Modify team member details in the Team section
3. Add actual event dates and details in the Events section
4. Update social media links in the footer
5. Customize colors in `styles.css` by modifying CSS variables
